## Setup

- Clone source
- npm install or yarn install

## IOS

After yarn install

- CMD: cd ios / pod install

## RUN

- yarn android or yarn ios

## Features

- Fetch token [v]
- Get me [v]
- Show list invoice [v]
- Filter: fromDate,toDate, orderBy, keyword, status [v]
- Pagination [v]
- Create invoice [v]
- After created invoice will shown in list [v]
- Use redux - redux toolkit [v]
